# Набор тест-кейсов: дымовое тестирование

Набор содержит минимальный комплект проверок, подтверждающих работоспособность критически важных сценариев приложения.

## TC-SMOKE-01
| Поле | Значение |
| --- | --- |
| **Название** | Успешный запуск приложения |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Браузер поддерживается спецификацией. |
| **Шаги** | 1. Открыть приложение в браузере.<br>2. Дождаться завершения загрузки. |
| **Ожидаемый результат** | Приложение открывается без критических ошибок, стартовый экран доступен для работы. |

## TC-SMOKE-02
| Поле | Значение |
| --- | --- |
| **Название** | Наличие основных элементов интерфейса на стартовом экране |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Приложение успешно запущено. |
| **Шаги** | 1. Осмотреть стартовый экран приложения. |
| **Ожидаемый результат** | Отображаются игровое поле, кнопка «Новая игра», переключатель сложности, таймер, счётчик мин и блок статистики. |

## TC-SMOKE-03
| Поле | Значение |
| --- | --- |
| **Название** | Первый ход и переход партии в состояние «в процессе» |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Открыта новая партия. |
| **Шаги** | 1. Открыть любую скрытую клетку.<br>2. Проверить состояние партии и таймер. |
| **Ожидаемый результат** | Открывается безопасная клетка, партия переходит в состояние «в процессе», таймер начинает отсчёт. |

## TC-SMOKE-04
| Поле | Значение |
| --- | --- |
| **Название** | Установка и снятие флага |
| **Тип тестирования** | Дымовое |
| **Предусловия** | На поле есть скрытая клетка, приложение открыто на настольном устройстве или включён мобильный режим «Флаг». |
| **Шаги** | 1. Установить флаг на скрытую клетку.<br>2. Снять флаг с той же клетки. |
| **Ожидаемый результат** | Флаг успешно устанавливается и снимается, счётчик мин обновляется корректно. |

## TC-SMOKE-05
| Поле | Значение |
| --- | --- |
| **Название** | Смена уровня сложности и создание нового поля |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Приложение запущено. |
| **Шаги** | 1. Выбрать другой уровень сложности.<br>2. Проверить параметры нового поля. |
| **Ожидаемый результат** | Создаётся новое поле выбранного уровня сложности, таймер сбрасывается, приложение остаётся работоспособным. |

## TC-SMOKE-06
| Поле | Значение |
| --- | --- |
| **Название** | Завершение партии поражением |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Идёт активная партия, доступна клетка с миной. |
| **Шаги** | 1. Открыть клетку с миной.<br>2. Проверить отображение результата. |
| **Ожидаемый результат** | Партия завершается поражением, отображается сообщение о поражении, мины раскрыты, дальнейшие действия по полю заблокированы. |

## TC-SMOKE-07
| Поле | Значение |
| --- | --- |
| **Название** | Обновление и сохранение статистики завершённой партии |
| **Тип тестирования** | Дымовое |
| **Предусловия** | Завершена хотя бы одна партия победой или поражением. |
| **Шаги** | 1. Зафиксировать текущие значения статистики.<br>2. Перезагрузить страницу.<br>3. Проверить блок статистики после загрузки. |
| **Ожидаемый результат** | Статистика завершённых партий отображается корректно и сохраняется после перезагрузки страницы. |
