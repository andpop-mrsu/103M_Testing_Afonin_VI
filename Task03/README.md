# Task03

В каталоге собраны наборы тест-кейсов для веб-игры "Saper", подготовленные по чек-листам из `Task02`.

- `positive.md` - pozitivnye scenarii funkcionalnogo testirovaniia.
- `negative.md` - negativnye scenarii funkcionalnogo testirovaniia.
- `non-functional.md` - nefunkcionalnye test-keisy.
- `smoke_test.md` - nabor dymovykh testov.

Vse dokumenty oformleny v Markdown i opiraiutsia na specifikaciiu iz `Task01/spec.docx` i chek-listy iz `Task02`.
